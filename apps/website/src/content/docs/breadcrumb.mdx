# Breadcrumb

面包屑导航组件，用于显示当前页面在层级结构中的位置，帮助用户快速了解和跳转到上级页面。

## Import

```tsx
import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbPage,
  BreadcrumbSeparator,
  BreadcrumbEllipsis,
} from '@gedatou/ui'
```
