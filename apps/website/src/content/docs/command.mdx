# Command

基于 cmdk 构建的命令面板组件，支持快速搜索、键盘导航和分组展示。

## Import

```tsx
import {
  Command,
  CommandDialog,
  CommandInput,
  CommandList,
  CommandEmpty,
  CommandGroup,
  CommandItem,
  CommandShortcut,
  CommandSeparator,
} from '@gedatou/ui'
```

将 Command 嵌入对话框中，适用于全局命令面板场景。

如需显式关闭按钮，请额外从 `@gedatou/ui` 导入 `DialogContentClose` 并在 `children` 中组合。
