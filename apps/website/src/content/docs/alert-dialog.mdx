# AlertDialog

模态确认对话框，用于需要用户明确确认的破坏性或重要操作。基于 Radix UI AlertDialog 构建。

## Import

```tsx
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogMedia,
  AlertDialogTitle,
  AlertDialogTrigger,
} from '@gedatou/ui'
```
