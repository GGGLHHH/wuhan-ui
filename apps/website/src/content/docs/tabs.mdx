# Tabs

用于在不同内容面板之间切换的选项卡组件，基于 Radix UI Tabs 构建，支持水平和垂直方向。

## Import

```tsx
import { Tabs, TabsList, TabsTrigger, TabsContent } from '@gedatou/ui'
```

## Usage

</ComponentPreview>
