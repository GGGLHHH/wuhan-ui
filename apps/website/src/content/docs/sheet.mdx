# Sheet

侧边抽屉面板组件，从屏幕边缘滑出，用于展示补充内容或表单。基于 Radix UI Dialog 构建。

## Import

```tsx
import {
  Sheet,
  SheetClose,
  SheetContent,
  SheetDescription,
  SheetFooter,
  SheetHeader,
  SheetTitle,
  SheetTrigger,
} from '@gedatou/ui'
```
