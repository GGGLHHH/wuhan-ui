# Table

用于展示结构化数据的表格组件，支持表头、表体、表尾和标题。

## Import

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableFooter,
  TableHead,
  TableRow,
  TableCell,
  TableCaption,
} from '@gedatou/ui'
```

## Usage

<ComponentPreview code={TableDemoSource}>
  <TableDemo />
</ComponentPreview>
