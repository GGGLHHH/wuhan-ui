# Chart

基于 Recharts 的图表组件，提供统一的主题配置、工具提示和图例支持。

## Import

```tsx
import {
  ChartContainer,
  ChartTooltip,
  ChartTooltipContent,
  ChartLegend,
  ChartLegendContent,
  type ChartConfig,
} from '@gedatou/ui'
```

### ChartConfig

```tsx
type ChartConfig = {
  [key: string]: {
    label?: ReactNode
    icon?: React.ComponentType
  } & (
    | { color?: string; theme?: never }
    | { color?: never; theme: Record<'light' | 'dark', string> }
  )
}
```
