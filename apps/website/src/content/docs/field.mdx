# Field

表单字段组件集合，提供标签、描述、错误提示等表单布局能力，支持垂直、水平和响应式排列方向。

## Import

```tsx
import {
  Field,
  FieldContent,
  FieldDescription,
  FieldError,
  FieldGroup,
  FieldLabel,
  FieldLegend,
  FieldSeparator,
  FieldSet,
  FieldTitle,
} from '@gedatou/ui'
```

### FieldGroup

字段分组容器，用于包裹多个 Field，提供统一间距。

### FieldSet

基于 `<fieldset>` 的语义化分组容器。

### FieldContent

字段内容容器，用于包裹输入控件、描述和错误提示。

### FieldTitle

字段标题组件，适用于不需要 `<label>` 语义的场景。

### FieldSeparator

字段分隔线，可选传入 children 作为分隔文本。
