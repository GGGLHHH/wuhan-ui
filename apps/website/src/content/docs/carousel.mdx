# Carousel

基于 Embla Carousel 的轮播组件，支持水平和垂直方向滚动、键盘导航和插件扩展。

## Import

```tsx
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselPrevious,
  CarouselNext,
} from '@gedatou/ui'
```

## Usage

</ComponentPreview>
