# Item

用于构建列表项、信息项或设置项的组合式组件，支持媒体区、标题、描述和操作区等常见布局。

## Import

```tsx
import {
  Item,
  ItemActions,
  ItemContent,
  ItemDescription,
  ItemGroup,
  ItemHeader,
  ItemFooter,
  ItemMedia,
  ItemSeparator,
  ItemTitle,
} from '@gedatou/ui'
```
