# Direction

文本方向提供者组件，基于 Radix UI 的 Direction 原语，用于控制子组件的 LTR/RTL 布局方向。

## Import

```tsx
import { DirectionProvider, useDirection } from '@gedatou/ui'
```

## Usage

<ComponentPreview code={DirectionDemoSource}>
  <DirectionDemo />
</ComponentPreview>

### useDirection

返回当前上下文中的文本方向值的 Hook。

```tsx
const direction = useDirection()
// direction: 'ltr' | 'rtl'
```
