# Tooltip

工具提示组件，当用户悬停或聚焦元素时显示简短的信息提示。基于 Radix UI Tooltip 构建。

## Import

```tsx
import { Tooltip, TooltipContent, TooltipProvider, TooltipTrigger } from '@gedatou/ui'
```

## Usage

<ComponentPreview code={TooltipDemoSource}>

</ComponentPreview>
