# Collapsible

可折叠内容区域组件，基于 Radix UI Collapsible 构建，支持展开和收起内容。

## Import

```tsx
import { Collapsible, CollapsibleTrigger, CollapsibleContent } from '@gedatou/ui'
```

## Usage

<ComponentPreview code={CollapsibleDemoSource}>

</ComponentPreview>
