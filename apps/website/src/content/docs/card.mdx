# Card

用于承载标题、描述、内容和操作区域的基础卡片容器组件。

## Import

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardAction,
  CardContent,
  CardFooter,
} from '@gedatou/ui'
```
