# ToggleGroup

切换按钮组组件，允许用户在一组选项中进行单选或多选切换。基于 Radix UI ToggleGroup 构建。

## Import

```tsx
import { ToggleGroup, ToggleGroupItem } from '@gedatou/ui'
```

## Usage

<ComponentPreview code={ToggleGroupDemoSource}>

</ComponentPreview>
