# NavigationMenu

用于网站主导航的菜单组件，支持下拉内容面板和键盘导航。基于 Radix UI 构建。

## Import

```tsx
import {
  NavigationMenu,
  NavigationMenuList,
  NavigationMenuItem,
  NavigationMenuTrigger,
  NavigationMenuContent,
  NavigationMenuLink,
} from '@gedatou/ui'
```
