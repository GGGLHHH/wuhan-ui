# Select

下拉选择器组件，允许用户从一组预定义选项中选择一个值。基于 Radix UI Select 构建。

## Import

```tsx
import {
  Select,
  SelectContent,
  SelectGroup,
  SelectItem,
  SelectLabel,
  SelectTrigger,
  SelectValue,
} from '@gedatou/ui'
```
