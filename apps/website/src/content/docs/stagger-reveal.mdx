# StaggerReveal

让被包裹的子元素从下往上依次出现的动画组件。首批可见元素会按顺序交错出现，后续滚动进入的元素即时响应。基于 [Motion](https://motion.dev) 实现。

## Import

```tsx
import { StaggerReveal } from '@gedatou/ui'
```

</ComponentPreview>
