# Popover

弹出框组件，点击触发器后在其附近显示浮动内容面板。基于 Radix UI 构建。

## Import

```tsx
import {
  Popover,
  PopoverTrigger,
  PopoverContent,
  PopoverHeader,
  PopoverTitle,
  PopoverDescription,
  PopoverAnchor,
} from '@gedatou/ui'
```
