# InputGroup

输入框组合组件，支持在输入框前后添加图标、文本、按钮等附加元素。

## Import

```tsx
import {
  InputGroup,
  InputGroupAddon,
  InputGroupButton,
  InputGroupInput,
  InputGroupText,
  InputGroupTextarea,
} from '@gedatou/ui'
```
