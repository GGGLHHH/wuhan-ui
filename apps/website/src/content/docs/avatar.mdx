# Avatar

用于展示用户头像的组件，支持图片、文字回退、徽标、尺寸变体和分组显示。基于 Radix UI Avatar 原语构建。

## Import

```tsx
import {
  Avatar,
  AvatarImage,
  AvatarFallback,
  AvatarBadge,
  AvatarGroup,
  AvatarGroupCount,
} from '@gedatou/ui'
```

## Usage

</ComponentPreview>
