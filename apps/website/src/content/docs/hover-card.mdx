# HoverCard

鼠标悬停时显示的弹出卡片组件，适用于展示链接或元素的预览信息。基于 Radix UI 构建。

## Import

```tsx
import { HoverCard, HoverCardTrigger, HoverCardContent } from '@gedatou/ui'
```

## Usage

<ComponentPreview code={HoverCardDemoSource}>

</ComponentPreview>
