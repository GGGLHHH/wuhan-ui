# Resizable

可调整大小的面板组组件，支持水平和垂直方向拖拽调整，基于 react-resizable-panels 构建。

## Import

```tsx
import { ResizablePanelGroup, ResizablePanel, ResizableHandle } from '@gedatou/ui'
```

## Usage

<ComponentPreview code={ResizableDemoSource}>

</ComponentPreview>
