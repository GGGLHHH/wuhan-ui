# Accordion

可折叠的内容面板组件，基于 Radix UI Accordion 构建，支持单选和多选模式。

## Import

```tsx
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from '@gedatou/ui'
```

## Usage

<ComponentPreview code={AccordionDemoSource}>
  <AccordionDemo />
</ComponentPreview>
