# Menubar

水平菜单栏组件，基于 Radix UI 构建，支持子菜单、复选项、单选项和快捷键提示。

## Import

```tsx
import {
  Menubar,
  MenubarMenu,
  MenubarTrigger,
  MenubarContent,
  MenubarItem,
  MenubarSeparator,
  MenubarShortcut,
  MenubarCheckboxItem,
  MenubarRadioGroup,
  MenubarRadioItem,
  MenubarSub,
  MenubarSubTrigger,
  MenubarSubContent,
  MenubarLabel,
  MenubarGroup,
  MenubarPortal,
} from '@gedatou/ui'
```
