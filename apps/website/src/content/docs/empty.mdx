# Empty

空状态组件，用于在列表或页面没有数据时展示占位提示信息和操作入口。

## Import

```tsx
import {
  Empty,
  EmptyHeader,
  EmptyMedia,
  EmptyTitle,
  EmptyDescription,
  EmptyContent,
} from '@gedatou/ui'
```

## Usage

</ComponentPreview>
