# InputOTP

一次性密码（OTP）输入组件，支持分组和分隔符，适用于验证码输入场景。基于 input-otp 构建。

## Import

```tsx
import { InputOTP, InputOTPGroup, InputOTPSlot, InputOTPSeparator } from '@gedatou/ui'
```

## Usage

</ComponentPreview>
