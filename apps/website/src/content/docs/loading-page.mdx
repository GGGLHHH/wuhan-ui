# LoadingPage

基于 Lottie 的加载动画组件，使用 `@lottiefiles/dotlottie-react` 渲染。

## Import

```tsx
import { LoadingPage } from '@gedatou/ui'
```

## Usage

<ComponentPreview code={LoadingPageDemoSource}>
  <LoadingPageDemo />
</ComponentPreview>

## API Reference

该组件内部固定使用内置 Lottie 动画资源，`src`、`loop`、`autoplay` 为硬编码值，不可通过 props 覆盖。

其余 `DotLottieReactProps` 属性（如 `className`、`style` 等）可通过 props 透传。
