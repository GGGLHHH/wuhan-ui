# Combobox

可搜索的下拉选择组件，基于 Base UI Combobox 构建，支持筛选、清除、分组和多选标签模式。

## Import

```tsx
import {
  Combobox,
  ComboboxInput,
  ComboboxContent,
  ComboboxList,
  ComboboxItem,
  ComboboxEmpty,
  ComboboxGroup,
  ComboboxLabel,
  ComboboxSeparator,
  ComboboxChips,
  ComboboxChip,
  ComboboxChipsInput,
} from '@gedatou/ui'
```
