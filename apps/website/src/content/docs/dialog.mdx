# Dialog

模态对话框组件，用于在当前页面上方展示重要内容或需要用户交互的信息。基于 Radix UI Dialog 构建。

## Import

```tsx
import {
  Dialog,
  DialogContent,
  DialogContentClose,
  DialogDescription,
  DialogFooter,
  DialogFooterClose,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from '@gedatou/ui'
```
