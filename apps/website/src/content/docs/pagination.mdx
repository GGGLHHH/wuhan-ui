# Pagination

分页导航组件，用于在多页内容之间切换。支持上一页、下一页、页码和省略号。

## Import

```tsx
import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationPrevious,
  PaginationNext,
  PaginationEllipsis,
} from '@gedatou/ui'
```
