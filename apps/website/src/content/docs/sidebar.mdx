# Sidebar

侧边栏导航组件，提供应用级别的导航结构。支持展开/折叠、移动端适配、图标模式等多种交互方式。

## Import

```tsx
import {
  Sidebar,
  SidebarContent,
  SidebarGroup,
  SidebarGroupContent,
  SidebarGroupLabel,
  SidebarMenu,
  SidebarMenuButton,
  SidebarMenuItem,
  SidebarProvider,
} from '@gedatou/ui'
```
