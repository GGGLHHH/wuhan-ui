# RadioGroup

单选按钮组组件，允许用户从一组选项中选择一个，基于 Radix UI 构建。

## Import

```tsx
import { RadioGroup, RadioGroupItem } from '@gedatou/ui'
```

## Usage

<ComponentPreview code={RadioGroupDemoSource}>

</ComponentPreview>
